JRGrep README

What is JRGrep?

This Java application is basically a recursive file finder. You can search for files based on their filename, their contents, or both.

Requirements

The system must have a Java runtime version 1.4 or better installed. See http://java.sun.com/ for acquiring a runtime environment.

Using JRGrep

Start the program by invoking the java command like so:

java -jar jrgrep.jar

This will launch the program and from there you can utilize the graphical user interface to perform the search. The application is reasonably intuitable. The file name and pattern text fields expect regular expressions. If you are not familiar with regular expressions, search the web or read the book "Mastering Regular Expressions" by Jeffrey E. F. Friedl. Basically, typing a word or phrase will work.

Command-line Interface:

JRGrep sports a command-line interface, accessible through the com.bluemarsh.jrgrep.tty class. Use '-h' to access the help.

Building JRGrep

To compile the program from source code, you will need the JRGrep source package and Apache Ant. With Ant installed and the JRGrep source extracted to the current directory, type the following to compile the source code:

ant compile
